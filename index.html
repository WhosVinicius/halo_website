The halo editor.

Edit your code and texts in a lightweight and fast fashion

Installation:

Download the file below.
After it is downloaded, extract it into program files (x86).
The next step is adding the folder to Path.
To add the folder to path, search "path" in the windows search prompt. Then click the "Edit enviroment variables" and then on enviroment variables.
Then search for Path, click it twice, and then on New. Then add the directory path to the halo editor/release folder, and it's done!

Usage:

After adding the executable to path, you can run it via the terminal, like this:

halo "example.txt"

After opening the file, you will notice that you can't type anything. That is because the editor opens in Normal Mode, a mode for navigating the file. You can move the cursor with the WASD keys, or the keyboard arrow keys to move 5 lines down or up a time.

Commands:

A list of the commands avaible in halo:

These commands work in both Edit and Normal Mode:

ALT + Q => Quits the editor. (If the buffer is not saved, all progress will be lost, there will be no prompt)

CTRL + S => Saves the current buffer.

CTRL + ; => Adds // to the start of a line. (Comments for C, C++, JS,TS,Rust and C#)

TAB => Indents a line in two spaces.

CTRL + Backspace => Removes one indent from the line.

These commands are for Normal Mode only:

I => Enters insert mode.

U => Moves the line up.

H => Moves the line down.

X => Removes the line.

M => Moves the cursor to the start of the file.

N => Moves the cursor to the end of the file

Q => Moves the cursor to the start of the line.

E => Moves the cursor to the end of the line

For exiting Insert Mode, just press ESC
